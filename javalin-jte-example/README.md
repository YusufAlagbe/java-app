# Javalin JTE Example

This example demonstrates how to use [JTE (Java Template Engine)](https://jte.gg/) with Javalin 7.

## Running the example

```bash
mvn compile exec:java -Dexec.mainClass="app.JavalinJteExampleApp"
```

Then visit http://localhost:7070/
